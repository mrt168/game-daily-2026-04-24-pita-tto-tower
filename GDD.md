# GDD: ピタッとタワー

## コンセプト
- ジャンル: action
- コアループ: 左右に揺れるクレーンからブロックを落とし、前のブロックと揃えて高く積む
- 対象: ちょっとした暇つぶしをしたい人、タイミングゲームや物理パズルが好きな人
- プレイ時間: 1プレイ30秒〜3分

## MDA分析
- **Mechanics（メカニクス）**: ワンタップでクレーンからブロック落下、重力とブロックの重なり判定、色・サイズ変化、コンボ倍率
- **Dynamics（ダイナミクス）**: 「ちょうどいいタイミングで落とす」緊張感、積むほど揺れ幅が広がる難化、ミスすると「一発即死」の明確なリスク
- **Aesthetics（美学）**: Challenge（挑戦）、Sensation（瞬間的な達成感）、Expression（自分のタワーの高さ自慢）

## ゲームフロー
1. タイトル画面（ロゴ、タップでスタート） → 2. **チュートリアル**（初回のみ自動、2回目以降はスキップ可） → 3. ゲームプレイ → 4. タワー崩壊 → 5. リザルト（階数・ベストスコア・リトライ・シェア）

## 操作方法
- 画面のどこかをタップ / クリックすると、クレーンから吊り下げられたブロックが真下に落下する
- ただそれだけ。ワンタップで完結する

## ゲーム要素
1. **揺れるクレーン**: 画面上部を左右に揺れ動くクレーン。タップでブロック落下。
2. **ブロックの重なり判定**: 前ブロックとの重なり幅でスコア倍率を決定。完全に重なる「パーフェクト」ではコンボ発動。
3. **はみ出しペナルティ**: はみ出した部分は切り落とされ、次のブロックが小さくなる。ブロック幅が0になったらゲームオーバー。
4. **コンボシステム**: 3回連続パーフェクトで幅ボーナス、x2/x4/x8 のスコア倍率。
5. **難化カーブ**: 階数が上がるごとにクレーン揺動速度が上昇、揺動幅も微増。
6. **カメラスクロール**: タワーが高くなるとカメラが上にスクロール、下のブロックが見えなくなる（迫力演出）。
7. **色のグラデーション**: 階数ごとにブロックの色がHSV回転、美しい虹タワーが形成される。
8. **ハイスコア保存**: localStorageに best_height, best_score を保存。
9. **画面シェイクとパーティクル**: パーフェクトで星パーティクル、ミスでカメラシェイク。
10. **SNSシェア**: リザルト画面でCanvasスクショを生成してTwitter共有。

## 難易度曲線
- レベル1-3 (1-3階): クレーン速度 slow、ブロックは大きめ（成功体験）
- レベル4-7 (4-10階): 徐々にクレーン速度アップ（新要素: パーフェクト演出）
- レベル8+ (11階以降): クレーン速度最大、ブロック幅が自然に小さくなる（上級者向け）

## 視覚デザイン
- **カラーパレット**: 
  - 背景: #1a1a2e → #0f3460 グラデーション（夜空風）
  - ブロック: HSV回転で階数ごとに色変化（#f39c12, #e74c3c, #9b59b6, #3498db, #2ecc71...）
  - アクセント: #f1c40f (パーフェクト時の黄金)
- **スタイル**: minimal（ミニマル・モダン）。余計な装飾なし、幾何学のみ。
- **パーティクル効果**: パーフェクト時に星型パーティクル、ミス時に破片飛散。

## チュートリアル設計（必須）
### 教える内容（この順番で）
1. **基本操作**: 「画面をタップしてブロックを落とそう！」
2. **ゲーム目的**: 「前のブロックと重なるよう積み上げよう」
3. **失敗条件**: 「はみ出すとブロックが小さくなり、ゼロで終了」
4. **コツ**: 「ピッタリ重ねるとボーナス！連続で大爆発」

### チュートリアル方式
- インタラクティブ形式（実際にタップさせる）
- 矢印・ハイライトで操作箇所を示す
- 各ステップは1文+操作で完結
- 2回目以降はスキップ可能（localStorage: tutorial_done）

## サウンドデザイン
- BGM: ローファイ系ゆったり（Web Audio API でプログラム生成）
- SE: タップ音（クリック）、着地音（ボス）、パーフェクト音（チャイム）、崩壊音（ガラガラ）

## スコアシステム
- **基本ポイント**: 階ごと +100
- **重なり率ボーナス**: 重なり率 × 50 追加
- **パーフェクトボーナス**: +500 (コンボで ×2, ×4, ×8 増加)
- **ハイスコア保存**: localStorage (`best_height`, `best_score`)

## バイラル戦略（Grill Council結論）

### TOP3 採用戦略
1. **リザルト画面でワンタップTwitter共有**（スクショ+定型文+ハッシュタグ#ピタッとタワー）
2. **階数別称号のアンロック演出**（10階=ルーキー、30階=名匠、50階=神）
3. **毎回異なる色グラデーションタワー**（HSV回転で階ごとに色変化、絵画のように美しい）

### バイラル仕掛け実装方針
- GameOverScene にシェアボタン配置（target="_blank"のTwitter Web Intent）
- 10/30/50階到達時に画面中央で称号アンロック演出（Text + Tween + ParticleEmitter）
- 各階のブロック色は (階数 * 30) % 360 の HSV hue で決定
